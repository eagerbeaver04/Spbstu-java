Сборка приложения

Требования:
Java Development Kit (JDK) версии 8 или выше.

Шаги сборки:
- Откройте терминал (командную строку).
- Выберите директорию установки
- Используйте команду Git для клонирования репозитория:
  git clone https://github.com/eagerbeaver04/Spbstu-java.git
- Перейдите в каталог проекта
- Перейдите в каталог src
- Используйте команду Javac для компиляции Java-кода:
  javac Main.java
